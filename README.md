# Projet de cours : Computer Graphics

## Structure du projet

Le projet est divisé en **3 dossiers** couvrant l'ensemble des **4 TPs** :

* **Dossier TP1** : Contient le code du **TP1** ainsi que celui du **TP2** (qui est inclus dans ce même projet).
* **Dossier TP3** : Contient le code du TP3.
* **Dossier TP4** : Contient le code du TP4.

## Installation et Compilation

Ce projet utilise **CMake**. 

### Instructions étape par étape

Pour compiler l'un des TPs, placez-vous dans le dossier correspondant et exécutez les commandes suivantes dans votre terminal :

1. **Créer le dossier de build :**
   ```bash
   mkdir build
   ```

2. **Compiler le projet :**
   ```bash
   cd build
   cmake ..
   make
   ```
3. **Exécuter le projet :**
  ```bash
  ./opengl_program
  ```
